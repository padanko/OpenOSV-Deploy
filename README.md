# OpenOSV
無料で使える簡易掲示板システムです。<br>
まずは触りたい人は→<a href="https://padanko.github.io/">こちら</a>

**本アプリケーションはApache2などでリバースプロキシとして転送先に設定する場合を想定しています。**
<hr>

## まずはじめに
ビルドしたら、BBSフォルダを作成してください。<br>
その後staticフォルダを作成し、そこにjQueryを入れてください。<br>
